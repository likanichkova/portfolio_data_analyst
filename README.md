# game_industry_analysis
Анализ данных о видеоиграх
## Описание проекта

В проекте проведён анализ данных о видеоиграх. В датасете содержится информация о названиях игр, платформах, жанрах, годах выпуска, оценках пользователей и критиков, а также продажах в разных регионах.

Цель проекта — подготовить данные к анализу и выделить основные особенности игрового рынка.

## Навыки и инструменты

- Python
- pandas
- предобработка данных
- работа с пропусками
- проверка дубликатов
- изменение типов данных
- категоризация данных
- базовый анализ данных
- формулирование аналитических выводов

## Этапы работы

1. Загрузка и первичный обзор данных.
2. Проверка пропусков, дубликатов и типов данных.
3. Предобработка данных.
4. Добавление новых признаков.
5. Базовый анализ данных.
6. Формулирование итоговых выводов.

## Что было сделано

В ходе проекта были изучены данные о видеоиграх и выполнена подготовка датасета к дальнейшему анализу.

Основные шаги:
- приведены названия столбцов к единому стилю;
- изучены пропущенные значения;
- проверено наличие дубликатов;
- скорректированы типы данных;
- добавлены дополнительные признаки;
- сформулированы выводы по результатам обработки данных.

## Вывод

Проект демонстрирует базовые навыки аналитика данных: загрузку и изучение данных, предобработку, работу с пропусками и типами данных, создание новых признаков и оформление аналитических выводов.

Подготовленный датасет можно использовать для дальнейшего анализа игрового рынка: сравнения платформ, жанров, региональных продаж и оценок пользователей.
